# hr-attrition-analysis
HR Attrition Analysis project using Python, Pandas, and data visualization techniques to explore employee turnover patterns. Includes data cleaning, preprocessing, and insights to identify key factors affecting employee retention and improve decision-making.

The main objective of this project is to analyze employee data and understand the reasons behind employee attrition, helping organizations improve retention strategies.

---

1. Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

2. Dataset Information
The dataset contains employee-related details such as:
- Age  
- Department  
- Job Role  
- Salary  
- Years at Company  
- Job Satisfaction  
- Attrition Status  

---

3. Data Cleaning & Preprocessing
The following steps were performed:
- Handled missing values  
- Removed duplicate records  
- Converted data types where necessary  
- Encoded categorical variables  
- Cleaned inconsistent data  

---

4. Exploratory Data Analysis (EDA)
Key analyses performed:
- Attrition rate by department  
- Salary vs Attrition relationship  
- Age distribution of employees  
- Job satisfaction impact on attrition  
- Work experience vs attrition patterns  

---

5. Key Insights
- Employees with lower salary have higher attrition rates  
- Sales department shows the highest turnover  
- Younger employees are more likely to leave  
- Job satisfaction strongly impacts retention  
- Employees with fewer years in the company tend to leave more
